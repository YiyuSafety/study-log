# 我的学习记录

这里用来记录我的学习过程、笔记、截图证据和项目产出。

## 仓库结构
- daily/：每天的学习记录
- notes/：专题笔记
- screenshots/：学习截图和成果截图
- projects/：项目和学习产出

## 目前在学
- Python
- Linux
- Web 基础
- GitHub 基础

## 记录方式
我会把每天学了什么、做了什么、遇到了什么问题，记录到 daily 中；
把专题总结整理到 notes 中；
把截图证据放到 screenshots 中；
把真正能展示的成果放到 projects 中。

## 目标
把学习过程和成果整理成长期可查看、可展示、可复盘的路线图。
